# UniWay 🇰🇿

**Онлайн-платформа для карьерного старта студентов Казахстана**

## О проекте

UniWay — сервис подготовки к стажировкам и первой работе через реальные тестовые задания, AI-подготовку к собеседованиям и создание резюме с подтверждёнными навыками.

## ✅ Реализованные функции

### Страницы
- **Главная** — Hero, Features, How It Works, задания, компании, CTA
- **Каталог заданий** — с фильтрами по направлениям (IT, Маркетинг, Дизайн, Бизнес) и поиском
- **Страница задания** — детали, загрузка решения, AI-проверка
- **Регистрация / Вход** — роли студент/компания, демо-вход
- **О нас** — миссия, команда, ценности

### Кабинет студента
- **Дашборд** — статистика, активные задания, AI-собеседования, навыки
- **Профиль** — личная информация, направление
- **Мои задания** — выполненные и активные
- **AI-собеседование** — полная симуляция HR-интервью с оценкой 0-100
- **Портфолио** — выполненные работы, загрузка проектов
- **Резюме** — подтверждённые навыки, результаты заданий, AI-оценки

### Кабинет компании
- **Дашборд** — статистика, задания, топ-кандидаты
- **Задания** — управление тестовыми заданиями
- **Кандидаты** — таблица с баллами, AI-оценками, сортировкой
- **Профиль компании** — редактирование информации

### Панель администратора
- Статистика платформы
- Управление пользователями и заданиями

## 🤖 AI-модули

### AI-собеседование
- Симуляция HR-интервью по направлениям (IT, Маркетинг, Дизайн, Бизнес)
- 5-7 вопросов по выбранной специальности
- Мгновенная обратная связь после каждого ответа
- Итоговый балл 0-100
- Сохранение в профиль студента

### AI-проверка заданий
- Анализ загруженных решений
- Балл + комментарии
- Сильные стороны и ошибки

## 🏗️ Технический стек

- **Backend**: Hono (Cloudflare Workers)
- **Frontend**: Vanilla JS SPA + Tailwind CSS
- **Icons**: Font Awesome 6
- **Build**: Vite + Wrangler
- **Deploy**: Cloudflare Pages

## 🚀 Запуск

```bash
npm run build
pm2 start ecosystem.config.cjs
```
